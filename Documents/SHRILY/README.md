# SHRILY

Modern Python project structure with AI-assisted quality and automation.

## Project Structure

- `src/` - Main source code
- `core/`, `api/`, `infrastructure/` - Domain layers
- `tests/` - Unit and integration tests
- `scripts/` - Utility scripts
- `config/` - Configuration files
- `docs/` - Documentation

## Features
- Black, Ruff, Mypy, Pytest
- Pre-commit hooks
- GitHub Actions CI/CD
- .env.example and .gitignore
- VSCode settings and recommended extensions

## Getting Started
1. Install dependencies
2. Set up your environment variables
3. Run tests: `pytest`

## Automation & Quality
- Code formatting, linting, type checking, and tests automated via pre-commit and CI
